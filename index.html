📌 IMPORTANTE — Come aprire questo file

Questo dashboard non funziona nell'anteprima integrata (icona occhio 👁).
Devi aprirlo direttamente in Google Chrome:

  1. Trova il file dashboard_milionario.html nella cartella outputs
  2. Fai doppio clic sul file (si aprirà in Chrome)
  3. Oppure trascinalo nella barra degli indirizzi di Chrome

Se apri già il file in Chrome e vedi questo errore, controlla anche la condivisione del Google Sheet:

  1. Apri il tuo Google Sheet
  2. Clicca Condividi → in "Accesso generale" seleziona "Chiunque abbia il link → Visualizzatore"
  3. Clicca Fine, poi ricarica (F5)
